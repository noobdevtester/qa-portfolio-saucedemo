# QA Portfolio – Sauce Demo Website

## Project Overview
This project demonstrates manual testing of the Sauce Demo e-commerce web application.
The goal is to identify functional, UI, and validation issues while documenting test cases
and defects using standard QA practices.

## Application Under Test
Sauce Demo – E-commerce testing website

## Scope
- Login functionality
- Product listing
- Add to cart
- Checkout process
- Error handling

## Testing Types
- Functional testing
- Smoke testing
- Regression testing
- UI testing

## Tools Used
- Manual testing
- Jira-style defect reporting
- GitHub documentation
